## Develop
1. Install NodeJS
1. Install pnpm `npm install pnpm`
1. Install dependencies `pnpm install`
1. Rename `.env.example`  to `.env`, Edit `.env` set your bot id and api key
1. Dev `pnpm run dev`

## Deploy
 - pm2: `pnpm run start` 
 - docker: `./build_local_image.sh`
